# Playwright Web Automation & Scraping Project

This project demonstrates browser automation and web scraping using Python.

## Technologies Used

- Python
- Playwright
- BeautifulSoup
- Pandas

## Features

- Automated login using Playwright
- Form filling automation
- Page navigation
- Screenshot capture
- Web scraping using BeautifulSoup
- Data processing using Pandas
- CSV export
- PDF report generation

## Project Workflow

1. Launch browser using Playwright
2. Open login page
3. Fill username and password automatically
4. Navigate to quotes page
5. Extract quotes data
6. Convert extracted data into Pandas DataFrame
7. Save data to CSV file
8. Capture screenshots
9. Generate PDF report

## How to Run

### Install dependencies

pip install -r requirements.txt

### Install Playwright browser

playwright install

### Run script

python main.py

## Output

The script generates:

- quotes_data.csv
- report.pdf
